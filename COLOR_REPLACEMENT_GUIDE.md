# Color Replacement Guide

This document tracks the replacement of purple/indigo colors with the new blue color scheme:

- **Primary Blue**: #A0CEFD (replaces purple-600, indigo-600)
- **Light Blue**: #E4F2FF (replaces purple-100, indigo-100, purple-50)

## Usage in Tailwind Classes

For gradients: Use inline styles with `style={{ background: 'linear-gradient(to right, #A0CEFD, #E4F2FF)' }}`

For solid colors: Use `style={{ backgroundColor: '#A0CEFD' }}` or `style={{ color: '#A0CEFD' }}`

For text colors: Use `style={{ color: '#A0CEFD' }}` or `text-[#A0CEFD]`

For background colors: Use `style={{ backgroundColor: '#E4F2FF' }}` or `bg-[#E4F2FF]`
